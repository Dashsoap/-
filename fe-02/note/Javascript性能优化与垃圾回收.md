# 垃圾回收

## 常见的GC算法

### 引用计数
#### 核心思想：设置引用数，判断当前引用数是否为0
引用关系改变时 修改引用数字 被引用+1 被清空-1

当引用数字为0时  回收

优点：发现垃圾，立即回收。
最大限度减少程序暂停
缺点：无法清除循环引用
耗时大
### 标记清除
#### 核心思想：标记和清楚两个阶段完成
1.便利所有对象找标记活动对象
2.遍历所有对象清楚没有标记的对象
3.收回内存
